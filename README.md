# 💪 Happiness & Fitness AI Chatbot
This project was developed as part of an academic mini-project to demonstrate the implementation of user interaction, front-end development, and basic artificial intelligence logic using JavaScript.


## 🌟 Features

• 💬 Interactive chat interface with modern dark UI
• 🧠 Smart keyword-based response system
• 🏋️ Workout routine suggestions
• 🥗 Nutrition and diet guidance
• 🧘 Mental wellness and mindfulness tips
• 🔥 Motivational coaching
• ⚡ Typing animation for realistic chatbot feel
• 🎯 Quick-reply buttons for faster interaction
• 📱 Fully responsive design


## 🧮 Bonus Feature — BMI Calculator

Users can calculate their Body Mass Index by entering:

Height (cm) and Weight (kg)

The chatbot calculates BMI instantly and classifies it as:
• Underweight
• Normal Weight
• Overweight
• Obese

This demonstrates the use of conditional logic and dynamic data processing.


## 🛠️ Tech Stack

• HTML5 — Structure
• CSS3 — Styling & Layout
• JavaScript — Logic & Chatbot Intelligence
• Font Awesome — Icons
• Google Fonts — Typography



## 🧠 How It Works

The chatbot uses a rule-based artificial intelligence approach:

1. User enters a message
2. JavaScript analyzes keywords
3. Message is categorized (Workout / Nutrition / Motivation / etc.)
4. A suitable response is selected from predefined datasets
5. Response is displayed with animation effects
